Основні команди GIT

Команда Значення
git config --global user.email
git config --global user.name		 Задання параметрів які мають співпадати з профілем GitHub
cls перегляд наявних файлів в директорії
ls очистка екрану терміналу
cd (назва папки) перехід в вказану директорію
cd .. повернення до попередньої директорії
git init ініціалізація git
git status перегляд статусу змін в файлах
git add . додавання для відслідкування git файлів
git rm --cached (назва файлу) зупинка слідкування за файл
git commit -m '' комміт змін файлів
git remote add origin https://github.com/boris2606/git-course.git( це приклад ) Привязка локального репозиторію до існуючого
git push -u origin master Переміщення з локального репозиторію до GitHub
git push Переміщення з локального репозиторію до GitHub певних змін в проекті
git clone (посилання на репозиторій) Клонування репозиторію на ПК за посиланням
git pull Копіювання останніх змін з серверу репозиторію на локальний GIT

Робота з гілками GIT

Команда Значення
git branch Перегляд гілки на якій розташовані, та перегляд гілок які наявні
git branch (назва) Створення гілки
git branch -D (назва) Видалення гілки
git checkout (назва) Перехід до необхідної гілки
git checkout -b (назва) Створення та перехід до нової гілки
git merge (назва гілки) Поєднання гілок (під гілкою мастер)
